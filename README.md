# PixivCollection

[预览](https://pixiv.orilight.top/)

## 简介

个人的P站收藏夹，瀑布流布局，带有简单的浏览和筛选功能

主要是为了防止作者删图跑路丢失数据

因为图片数据是一次性全部加载的，为了减轻卡顿自己瞎写了个虚拟列表 ~~但效率十分感人并没有什么软用~~

原始尺寸的图片是从 OneDrive 上加载的，加载慢、加载失败属于正常现象
